# Recipe book service

In progress...
Front off the app is in the different repo -> "recipes_front"

## As a user you will be able to: 
* register
* login
* edit your personal data
* browse throught different recipes
* add your own recipes
* update or remove your recipes

## Tech & framework used:
* Java 17
* Spring boot
* Rest API
* Lombok
* Spring Security/JWT
* JPA/Hibernate
* PostgreSQL
* Angular
* Bootstrap
